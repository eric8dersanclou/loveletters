# 写给你的信 - H5 版本 v2.0

一个温馨的信件应用，可以写给重要的人，支持分享功能。

## ✨ 功能特性

- 💌 **写信**：选择封面样式，写下心意
- 📬 **看信**：3D 信封开启动画，温馨体验
- 🔗 **分享**：信件数据编码到 URL，分享后对方可直接查看
- ✏️ **编辑**：支持修改已写的信件
- 🗑️ **删除**：可以删除不需要的信件
- 💾 **本地存储**：使用 localStorage 保存信件
- 📱 **响应式设计**：适配手机和电脑

## 🎨 视觉特性

- 🌸 **信纸背景**：精美的渐变和图案背景
- 💕 **爱心撒花**：拆开信封时的爱心爆发效果
- ✨ **彩纸飘落**：持续的彩纸动画效果
- 🖋️ **优雅字体**：使用「站酷小薇」和「Noto Serif SC」中文字体

## 🚀 部署到 GitHub Pages

### 步骤 1：创建 GitHub 仓库

1. 打开 [github.com](https://github.com)，登录你的账号
2. 点击右上角 **+** → **New repository**
3. 填写仓库信息：
   - **Repository name**: `letter-to-you`（或你喜欢的名字）
   - **Description**: 写给你的信 - H5版本
   - 选择 **Public**（公开仓库才能免费使用 Pages）
   - 勾选 **Add a README file**
4. 点击 **Create repository**

### 步骤 2：上传文件

在仓库页面：
1. 点击 **Add file** → **Upload files**
2. 上传这两个文件：
   - `index.html`
   - `app.js`
3. 点击 **Commit changes**

### 步骤 3：开启 GitHub Pages

1. 在仓库页面，点击 **Settings**（顶部标签）
2. 左侧菜单找到 **Pages**
3. **Source** 部分选择 **Deploy from a branch**
4. **Branch** 选择 **main**（或 master），文件夹选 **/(root)**
5. 点击 **Save**

### 步骤 4：等待部署完成

- 等待 1-2 分钟
- 刷新页面，会显示你的访问链接：
  ```
  https://你的用户名.github.io/letter-to-you/
  ```
- 点击链接即可访问

## 📤 分享给朋友

1. 打开应用，写一封信
2. 点击"分享"按钮复制链接
3. 把链接发给朋友
4. 朋友打开链接后，**无需登录，直接就能看到完整的信件内容！**

分享链接示例：
```
https://你的用户名.github.io/letter-to-you/?letter=eyJ0Ijoi...（编码后的数据）
```

## ✏️ 编辑信件

1. 打开已写的信件
2. 点击"编辑"按钮
3. 修改内容后点击"保存修改"

## 🗑️ 删除信件

1. 打开要删除的信件
2. 点击"删除"按钮
3. 确认删除后信件将被永久删除

## 💡 技术说明

- 分享链接包含完整的信件数据（Base64 编码）
- 数据存储在浏览器 localStorage 中
- 不同设备访问数据不互通（除非使用分享链接）
- 使用 Google Fonts 加载中文字体

## 📱 分享封面预览

分享链接时，会自动生成预览卡片：

- **微信**：显示标题、描述和封面图
- **QQ**：显示卡片式预览
- **微博**：显示大卡片预览
- **Twitter**：显示 Summary Card
- **Facebook**：显示链接预览

### 自定义分享封面图

如果你想使用自己的封面图片，可以：

1. 准备一张 1200×630 像素的图片
2. 上传到图床（如 GitHub、Imgur、SM.MS 等）
3. 修改 `index.html` 中的 `og:image` 和 `twitter:image` 链接

```html
<meta property="og:image" content="你的图片链接">
<meta name="twitter:image" content="你的图片链接">
```

推荐封面图主题：
- 💌 信封/信件相关
- 🌸 粉色/浪漫风格
- ✨ 温馨/温暖氛围
